Amazon Web Services

Infrastructure as a Service (IaaS) is the most flexible cloud computing model and allows for
automated deployment of servers, processing power, storage, and networking. IaaS clients have truer
control over their infrastructure than users of PaaS or SaaS services.
Amazon Web Services (AWS) is a secure cloud services platform, offering compute power,
database storage, content delivery and other functionality to help businesses scale and grow.

AWS is:

  • Cost-Effective
  • Elastic and Scalable
  • Secure
  • Flexible
