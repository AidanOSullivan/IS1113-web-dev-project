Gmail

Communications as a Service (CaaS) is an outsourced enterprise communications solution that can be leased from a single vendor.
Such communications can include voice over IP (VoIP or Internet telephony), instant messaging (IM), collaboration and
videoconference applications using fixed and mobile devices.

Advantages of gmail as a cloud service are:

  • No server required for the user
  • Easy to make your email mobile
  • Plenty of storage
  • Eamils are backed up to the secure Google web data system
